# Laboratorio de Sistemas Embebidos

## Laboratorio N° 1: Convertidor de Binario, Decimal, Octas y Hexagecimal usando PIC16F887.

### 1. Introducción
#En este laboratorio se desarrolló un sistema embebido basado en el microcontrolador PIC16F887 capaz de convertir un número binario de 4 bits (D, C, B, A) a sus representaciones en sistema decimal, octal y hexadecimal.
El sistema permite visualizar el resultado mediante dos displays de 7 segmentos, implementando un decodificador digital por software.
